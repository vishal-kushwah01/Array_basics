# Array Basics in Java

This repository contains beginner-friendly Java programs to understand fundamental array concepts.

## Topics Covered

* Array Declaration and Initialization
* Taking Input in Arrays
* Printing Array Elements
* Finding Largest Element in an Array
* Finding Smallest Element in an Array
* Linear Search
* Passing Arrays as Function Arguments

## Files

* basic.java
* Largest_Number.java
* Smallest_Number.java
* Linear_Search.java
* ArrayAsFunction.java

## Purpose

This repository is part of my Java and DSA preparation journey. The goal is to build strong programming fundamentals before moving to advanced data structures and algorithms.
