# 🚀 Array Basics in Java

A collection of Java solutions for fundamental array problems frequently asked in coding interviews and online assessments.

This repository is part of my Data Structures and Algorithms (DSA) learning journey, where I focus on understanding problem-solving techniques, writing clean code, and improving time complexity.

---

## 📚 Topics Covered

### ✅ Array Basics
- Find Largest Element
- Find Smallest Element
- Sum of Array Elements
- Average of Array Elements
- Reverse an Array
- Linear Search
- Binary Search
- Find Duplicate Element
- Find Missing Number

### ✅ Pair & Subarray Problems
- Pair Sum (Two Sum - Brute Force)
- Three Sum (Brute Force)
- Print All Subarrays
- Maximum Sum Subarray (Brute Force)

### ✅ Prefix Sum
- Prefix Sum Array
- Range Sum Queries
- Maximum Subarray using Prefix Sum

### ✅ Trapping Rain Water
- Brute Force Approach
- Prefix & Suffix Maximum Arrays
